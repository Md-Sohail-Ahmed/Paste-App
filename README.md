# Paste-App

**Paste-App** is a simple and lightweight web application for creating and storing text or code snippets locally. It’s designed to help users quickly paste, edit, and manage snippets with a clean and minimal UI.

## Features

- Create and save text/code snippets
- Edit existing pastes
- Delete pastes
- Local storage–based data persistence (no backend required)
- Clean and responsive UI
- Fast performance using React + Vite

## Quick Start / Installation

Follow these steps to run the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/Md-Sohail-Ahmed/Paste-App.git
cd Paste-App

# 2. Install dependencies
npm install

# 3. Run the development server
npm run dev
